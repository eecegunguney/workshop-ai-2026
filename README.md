# AI Coding Agents Workshop

Presentation material for the AI Coding Agents Workshop, published as a static web page.

## Live page

https://pia-team.github.io/workshop-ai-2026/

## Contents

The workshop is delivered as self-contained HTML files. No build step or
external dependencies are required to view them.

- `index.html` — the presentation deck (19 slides).
- `gorevler.html` — participant tasks (12 exercises). The closing slide of the
  deck links here via a QR code and short URL so participants can open it on
  their own devices.

## Local preview

Open `index.html` directly in a browser, or serve the directory locally:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Deployment

The page is hosted with GitHub Pages from the `main` branch. Any change pushed
to `main` is published automatically.
